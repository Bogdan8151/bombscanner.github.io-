# Об’єкти виявлення

Дізнайся, які саме об'єкти наша програма може розпізнати.

При кожному оновленні программи, кількість об'єктів буде збільшуватися.

# Снаряд

основний елемент артилерійського пострілу, призначений для ураження цілей

# Ракета

літальний апарат, що рухається в просторі за рахунок дії реактивного руху

# Граната

Граната один з видів вибухових боєприпасів, призначений для ураження живої сили противника осколками й ударною хвилею

# Протитанкова міна

інженерний боєприпас (міна), призначений для знищення або виведення з ладу танків і інших броньованих машин противника

# Авіабомба

один з видів авіаційних боєприпасів, що скидаються з літака чи іншого літального апарата
